#### <a name="config">介绍</a>

　　服务提供者是用来注册服务的，进行服务容器绑定，一个提供者可以注册多个服务。

#### <a name="config">服务提供者</a>

`config/app.php`
```php
'cookie' => [
    'prefix' => '',      // cookie 前缀 默认 ''
    'domain' => null,    // cookie 域   默认 null
],
```
