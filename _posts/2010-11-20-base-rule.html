    控制器规则

    　　控制器文件默认保存在：/apps/模块/controllers/ 目录下；文件名与类名统一；
    　　控制器类命名规则：第一个字母大写，后面跟 Controller ；控制器继承 Controller 类；
    /apps/default/controllers/DemoController.php 

    
	class DemoController extends Controller
	{}
    动作规则

    　　动作命名规则：全部小写，后面跟 Action ；
    /apps/default/controllers/DemoController.php 

    
	class DemoController extends Controller
	{

		function indexAction()
		{}

	}
    视图规则

    　　视图模板文件默认保存在：/apps/模块/views/ 目录下；可以创建子级目录；目录及文件名尽量小写（不做强制限制，怎么写的怎么用就行了）；
    /apps/default/views/index.php
    /apps/default/views/demo/index.php

    # models 及 class 规则

    　　models 与 class 目录文件及类名称，对大小写不做强制限制；不过在开发时；还是要尽量定义一种统一的风格；参考类自动加载章节；
