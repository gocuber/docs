    # 功能模块

    　　一般情况下，开发网站时，会有多个功能模块，如 前台，后台，其他testapp 等；

    # 模型及类库子目录

    　　可以为模型 models，类库 class 按功能创建子目录；

    /*

/models                          模型目录
|-- db
|-- uc

/class                           项目公用类库
|-- admin
|-- ucenter

*/
    # 临时文件、资源目录

    　　在开发应用项目时，会用到一些缓存文件，临时文件，log日志及静态资源 图片、css、js等；以下目录不是必要目录，可随意修改或删除；

    /*

/common                          公用函数配置等文件

/data                            用于存放一些缓存，临时文件，log日志等
|-- cache
|-- logs

/res                             用于存放网站图片、css、js等资源文件
|-- images
|-- css
|-- js

*/
