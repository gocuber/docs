创建第一个应用
创建第一个应用
  • # 创建控制器
  • # 创建视图
  • # 访问url
# 创建控制器

　　新建控制器文件：/app/default/controllers/DemoController.php
　　代码： 


	class DemoController extends Controller
	{

		function welcomeAction()
		{
			$hash = array('title', 'Hello World');

			$this->display('demo/welcome', $hash); // 赋值并调用视图模板
		}

	}
# 创建视图

　　新建视图模板文件：/app/default/views/demo/welcome.php
　　代码： 


	echo $title; //Hello World
# 访问url

　　访问 http://localhost/index.php/demo/welcome/ 会显示 Hello World
　　可以设置URL重写使你的URL更加友好，例如可以这样访问：
　　http://localhost/demo/welcome/ 只需设置URL重写就可以了，会在URL重写章节中详细说明。
