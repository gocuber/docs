COOKIE使用
COOKIE使用
  • # set() 写COOKIE
  • # get() 读取COOKIE
  • # del() 删除COOKIE
  • # setraw() 写COOKIE

　　CubePHP框架中使用 Cookie 类来操作COOKIE；

set() 写COOKIE
Cookie::set('key', 'value', 3600, '/', 'gocuber.com');
get() 读取COOKIE
Cookie::get('key');
del() 删除COOKIE
Cookie::del('key');
setraw() 写COOKIE

　　与 set() 几乎完全相同，不同的是 setraw() 不会对 COOKIE 值进行 urlencode() URL 编码；

Cookie::setraw('key', 'value', 3600, '/', 'gocuber.com');
