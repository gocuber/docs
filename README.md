# Cuber
—— 更好、更快、更强的PHP开发框架

## 下载与安装

```
git clone https://github.com/gocuber/cuber.git cuber
cd cuber
composer install
```

## 目录结构

目录|说明
:--------|:--------
|app | 应用目录
|-- Controllers | 控制器目录
|-- Libs | 类目录
|-- Models | Model目录
|-- views | 视图目录
|bootstrap|入口
|config|配置
|public|index
|config|配置
|route|路由
|storage|数据存储
|vendor|Composer
